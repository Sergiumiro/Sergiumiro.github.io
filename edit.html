📄 Для скачивания DOCX запустите сервер

Генерация DOCX требует локального Node.js сервера. Выполните одну из команд в терминале рядом с файлом docx-server.js:

# Установить зависимость (один раз):
npm install -g docx
# Запустить сервер:
node docx-server.js

Сервер запустится на http://127.0.0.1:3721 и будет работать пока открыт терминал. После запуска нажмите «Скачать DOCX» снова.
